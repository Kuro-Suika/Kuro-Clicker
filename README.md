# Kuro Clicker
A software used to make, record, customize and share macros. Its basically just a much better Tiny Task with more qol and features. These are for keyboard and mouse macros.

## How to Install
- Go to the most recent [release](https://github.com/Kuro-Suika/Kuro-Clicker/releases) and install the KuroClicker.zip
- Extract it wherever you want but make sure it stays in the folder it comes with (it generates files and folders for configs and macros)
- double click the Kuro Clicker.exe
Thats it

## Features
- Lists all macros and has a favorites and search bar filter
- Easy sharing macros with the Share and Import buttons
- Complete customization with keybinds and other things in the settings tab
- Has a custom Safe Fail so macros stop when other apps like discord are focused
- Has a Force Focus option to avoid pop-ups from breaking the macro
- Option to record Keyboard, mouse clicks, and mouse movement separately
- Multiple playback modes; Once, Loop, Toggle, and Hold.
- Optional playback speed modifier
- Timeline editor for macros so you can easily see what your macro does and edit each and every action separately
- Ability to chain macros together, so a specified macro will play right after one finishes
- You can set a keybind for multiple different macros to play them separately at will
- Action Log so you can see exactly what the app is doing
- Ability to pop-out the status bar for a minimalistic taskbar while your recording or playing macros
